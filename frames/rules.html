Rules of the Online Judge

In order for the online judge to test your program, it must be sent to the testing server, compiled, and executed. To avoid causing issues with the online judge, please respect the following rules:

  1. Do not submit any code that accesses a directory or file outside of the working directory of the program.
  2. Do not submit any code that accesses the internet, opens a socket, or establishes any external connection in any way.
  3. Do not submit any code that modifies, creates, or deletes a file other than the specified output file.
  4. Do not submit any code that creates shutdown hooks.
  5. Do not submit malicious code of any kind.

Violating these rules can result in damage to the server. Please use the judge responsibly.
